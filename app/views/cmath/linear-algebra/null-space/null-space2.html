Matrix multiplication that yields 0 matrix

The following matrix multiplication example is interesting in that it somehow produces a zero matrix:


In normal arithmatic, the only way of a number multiplication giving us 0 is when at least one of the numbers is 0.

But in our case, no matrix is 0.

How can that happen?

It's the power of null space!

When we look more carefully, the second matrix is the null space of the first matrix. So, it's equivalent of 0 in normal number multiplication.

Remember, null space knows how to make a matrix 0 matrix by working as coeffients for a given linear system.
